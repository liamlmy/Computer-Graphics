Help - User Instruction

Author: Mingyang Li

3D space of an imaginary world!

This is a 3D graphic world full with different objects.

As you can see in the website, some objects are automatically moving and spinning to some extent.

Besides, the color of the world is shading.

You can you use your mouse to drag in the region of graphic to change the two objects' angle

Also you can press some keys to implement different functions.

Keyboard up arrow

To press this key, you can spin up the camera.

Keyboard down arrow

To press this key, you can spin down the camera.

Keyboard left arrow

To press this key, you can spin left the camera.

Keyboard right arrow

To press this key, you can spin right the camera.

Keyboard <

To press this key, you can move forward the camera.

Keyboard >

To press this key, you can move backward the camera.

Keyboard w

To press this key, you can move up the camera.

Keyboard s

To press this key, you can move down the camera.

Keyboard a

To press this key, you can move left the camera.

Keyboard d

To press this key, you can move right the camera.

Keyboard [

To press this key, you can spin counter-clockwise the camera.

Keyboard ]

To press this key, you can spin clockwise the camera.

Keyboard /

To press this key, you can accelerate the speed of movement.

Keyboard t

To press this key, you can move up the roosters.

Keyboard g

To press this key, you can move down the roosters.

Keyboard f

To press this key, you can move left the roosters.

Keyboard h

To press this key, you can move right the roosters.

Keyboard r

To press this key, you can move forward the roosters.

Keyboard y

To press this key, you can move backward the roosters.

Keyboard u

To press this key, you can rotate counter-clockwise the roosters.

Keyboard j

To press this key, you can rotate clockwise the roosters.

Keyboard -

To press this key, you can narrow the proportion of the width and height of the camera.

Keyboard +

To press this key, you can magnify the proportion of the width and height of the camera.

Keyboard space

To press this key, you can change the camera to the airplane view, so that the camera is automatically moving.

Keyboard (

To press this key, you can accelerate the speed of the airplane.

Keyboard )

To press this key, you can decelerate the speed of the airplane.

Mouse click and drag

If you drag you mouse, one of the rooster will to some extent.
