Help - User Instruction

Author: Mingyang Li

3D space of an imaginary world in different shading!

This is a 3D graphic world full with different objects in different shading.

As you can see in the website, some objects are automatically moving and spinning to some extent.

Besides, the shading of the objects can be changed.

You can click on some buttons and selections to implement different functions.

Also you can press some keys to implement different functions.

Keyboard up arrow

To press this key, you can spin up the camera.

Keyboard down arrow

To press this key, you can spin down the camera.

Keyboard left arrow

To press this key, you can spin left the camera.

Keyboard right arrow

To press this key, you can spin right the camera.

Keyboard u

To press this key, you can change to phong lighting with phong shading.

Keyboard i

To press this key, you can change to blinn-phong lighting with phong shading.

Keyboard o

To press this key, you can change to phong lighting with gouraud shading.

Keyboard p

To press this key, you can change to blinn-phong lighting with gouraud shading.

Keyboard <

To press this key, you can move forward the camera.

Keyboard >

To press this key, you can move backward the camera.

Keyboard w

To press this key, you can move up the camera.

Keyboard s

To press this key, you can move down the camera.

Keyboard a

To press this key, you can move left the camera.

Keyboard d

To press this key, you can move right the camera.

Keyboard [

To press this key, you can spin the camera in counter-clockwise way.

Keyboard ]

To press this key, you can spin the camera in clockwise way.

Buttons

User could use the control the buttons below the graphics to switch on/off the headlight, userlight, and others.

Input blank

User could input the value to control the color of light.
